---
title: System Requirements
description: "Minimum Requirements for Ultramarine Linux"
---

Ultramarine's requirements are simple:

- 64bit processor (x86_64 or aarch64)
- Compatible firmware (EFI, BIOS, etc)
- 1GB RAM
- 16GB disk space

If your device isn't a normal x86_64 computer, check out the [Anywhere section](/docs/anywhere) for more information about booting Ultramarine.

These are the recommended specs for a system that can handle opening a browser or a handful of native apps. You CAN run Ultramarine on lower specs, but you may run into a sluggish or unusable system.

- 4GB of RAM (More recommended)
- 1st generation Intel Core (or Equivalent) or newer

Xfce Edition can run with as little as 2GB of RAM. We suggest a minimum of 8GB of memory for Plasma and GNOME Editions, and 4GB for Budgie. You can run any Edition on less memory than we recommend, but your mileage may vary.
